# javaIM
基于springMVC的聊天室应用程序  
Chat room application based on spring MVC

# 启动前修改（推荐使用idea)
> web\WEB-INF\applicationContext.xml 修改数据库连接参数
> src\org\siro\config\OSSConfig.java 修改阿里云oss key（不然无法发送图片）
